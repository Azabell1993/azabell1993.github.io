결혼식 초대 API

버전 1.0.0 | 최종 업데이트: 2025-02-04

소개

개발자를 위한 공식 결혼식 초대 API입니다! 본 API를 통해 Azabell1993의 결혼식에 참석하기 위한 정보를 확인하고 RSVP 요청을 보낼 수 있습니다.

엔드포인트

초대 요청을 수락하려면 다음 엔드포인트로 요청을 보내세요. 

POST /api/v1/wedding/rsvp
{
    "name": "박지우",
    "accept or not": "accept",
    "message": "감사합니다! 참석할게요!"
}

초대 수락하기

응답

RSVP가 성공적으로 완료되면 다음과 같은 응답을 받습니다. 

{
    "status": "success",
    "message": "초대에 응해주셔서 감사합니다!",
    "data": {
        "event_date": "2025-05-24 17:30 KST",
        "event_location": "서울 성북구 성북로23길 9, 리보아르"
    }
}
